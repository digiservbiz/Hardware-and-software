📋 Cheat Sheet — EHS

Exam traps, RAID, truth tables, IP formulas, storage speeds, drivers, commands.

⚠️ Exam Traps
RAID 0 provides data redundancy.
❌ FALSE. RAID 0 = striping ONLY. ZERO redundancy. Any 1 disk fails → ALL data on ALL disks permanently lost.
RAID 1 doubles your storage capacity.
❌ FALSE. RAID 1 = mirroring. You lose 50%. 2×1TB = only 1TB usable.
RAID 5 survives 2 disk failures.
❌ FALSE. RAID 5 = exactly 1 failure. Second failure = all data lost.
IP AND Mask gives broadcast address.
❌ FALSE. IP AND Mask = Network ID. Broadcast = IP OR (NOT Mask).
NOT 255.255.255.0 = 255.255.255.255
❌ FALSE. NOT 255.255.255.0 = 0.0.0.255. NOT inverts every bit: 255→0, 0→255.
XOR gives 1 when at least one input is 1.
❌ FALSE. That's OR. XOR = 1 ONLY when inputs are DIFFERENT. 1 XOR 1 = 0.
A webcam is an output device.
❌ FALSE. Webcam = INPUT — sends video TO the computer.
Any number AND 255 = 0.
❌ FALSE. Number AND 255 = original number. AND with 0 = always 0.
devmgmt.msc opens Disk Management.
❌ FALSE. devmgmt.msc = Device Manager. diskmgmt.msc = Disk Management.
A UPS powers equipment for hours.
❌ FALSE. UPS = minutes. For hours, use a generator.
RAID 10 needs only 2 disks minimum.
❌ FALSE. RAID 10 = minimum 4 disks.
UEFI has the same disk limit as BIOS.
❌ FALSE. UEFI+GPT = 9.4 ZB and 128 partitions. BIOS+MBR = 2.2 TB and 4 partitions.
DDR4 RAM fits in a DDR3 slot.
❌ FALSE. DDR generations are physically incompatible — different notch/pins.
32-bit Windows can use all 16 GB RAM.
❌ FALSE. 32-bit OS = max 4 GB addressable. Use 64-bit for more.
SLC SSDs wear out faster than TLC.
❌ FALSE. SLC has the MOST write cycles. More bits/cell = faster wear. SLC > MLC > TLC > QLC.
Jordy (Cursistengroep) can write to ICTmap on the NAS.
❌ FALSE. Cursistengroep = NO ACCESS to ICTmap at all. Only ICTgroep (Steven, Noa).

IP AND (NOT Mask)

IP OR (NOT Mask)

⌨️ Key Commands
devmgmt.msc
Device Manager — view/fix hardware drivers. ⚠️ = problem
diskmgmt.msc
Disk Management — partition and format drives
msinfo32
System Information — full hardware/OS overview, BIOS mode
regedit
Registry Editor (use with extreme caution!)
ipconfig
Show IP, subnet mask, gateway. /all adds MAC + DNS
ping [IP]
Test connectivity. No reply = unreachable or firewall
pnputil /enum-drivers
List all 3rd party drivers in Driver Store
pnputil /add-driver x.inf /install
Add and install a driver
pnputil /delete-driver oem1.inf /force
Force remove a driver
net use X: \\\\IP\\Share /user:Name *
Map network drive to NAS folder
net use * /d
Disconnect ALL mapped drives
driverquery
List all installed drivers via command prompt
🔑 Must-Know Terms
VEN_8086 = Intel
VEN_10DE=NVIDIA · VEN_1002=AMD · VEN_10EC=Realtek. Found in Device Manager → Hardware IDs.
Driver Store
C:\\Windows\\System32\\DriverStore\\FileRepository. Staging = copying driver here first.
Hot Spare
Standby RAID disk that auto-rebuilds on failure. No human action needed.
TRIM
Pre-erases unused SSD blocks → keeps write speed high over time.
AHCI
SATA mode enabling NCQ + hot-plugging. Always set in BIOS (not IDE mode).
XMP Profile
BIOS preset enabling RAM at rated speed. Without XMP: DDR4-3200 runs at 2133 MHz!
Dual Channel RAM
2 matched sticks in same-color slots = 2× bandwidth. 3 sticks = disables it!
M.2 NVMe
PCIe bus: up to 20 GB/s. M.2 SATA = same 600 MB/s as regular SATA III.
DSM (Synology)
DiskStation Manager — Synology's browser-based NAS OS. Access via NAS IP.
CSM Mode
UEFI emulates Legacy BIOS for MBR disks. Disable for Secure Boot.
driversdownloader.com
Identify hardware using VEN_XXXX&DEV_XXXX from Device Manager Hardware IDs.
NAS Class Config
ICTgroep (Steven, Noa): ICTmap R/W, Cursistenmap read-only. Cursistengroep (Jordy, Ahmed, Kim): Cursistenmap R/W, ICTmap NO ACCESS.
